Description

Once the controller is developed, it can be saved as a pickle object. An example submission can be found here. Use it as a guide to ensure your submission is formatted correctly.

Requirements

Failure to meet all requiremnts may cause the submission portal to raise an error.

  • Use Python version 3.11.
  • Use the package versions specified here. The pickle object needs to be loaded and tested in the web server, so failure to comply with specific versions may result in failure to load the pickle obeject. The fewer the external packages used the lower the likelihood of getting submission errors. If you require some specific version of a package like tensorflow==X, please email pr1u20@soton.ac.uk. We will install it within the web server.

If you get any error or have any questions please raise an issue in github or contact pr1u20@soton.ac.uk
